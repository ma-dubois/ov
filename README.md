OV
=========

About
---------

This tiny library is used to compute overlap in molecular modelling softwares. It seems that the author of this code is [Greg Landrum](https://github.com/greglandrum).

Get this library
---------
````bash
  git clone https://github.com/ma-dubois/ov
````

Dependencies
---------

none

Build
--------
````bash
  cd ov;
  mkdir build;
  cd build;
  cmake ../;
  make;
````
